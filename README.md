## 📖 项目简介

这是一套完整的 **A 股短线量化交易策略和工具集**，包含：

- **五指标共振模型** — MA / RSI / KDJ / MACD / 成交量
- **双轨扫描策略** — 主线方向精选 + 独立突破扫描
- **决策树系统** — 9 条规则，做或不做，二选一
- **止盈止损规则** — 三层止损 + 分批止盈 + 仓位管理
- **全市场扫描脚本** — 纯 Python 标准库，零依赖，开箱即用

所有技术指标（MA、RSI、KDJ、MACD、布林线）均为纯 Python 实现，数据源使用腾讯财经 HTTP 接口，无需任何第三方库。

## 🚀 快速开始

### 运行全市场扫描

```bash
git clone https://github.com/gamelife1314/a-stock-quant-trading.git
cd a-stock-quant-trading
python3 scripts/scanner.py
```

扫描脚本会：
1. 获取全市场 A 股列表（腾讯接口）
2. 批量拉取 K 线数据
3. 执行三层信号链路扫描
4. 输出扫描结果到控制台并保存 JSON

### 三层信号链路

| 层级 | 信号 | 条件 |
|------|------|------|
| 第一层 | **地量止跌** | 量<20日均量60% + 涨跌幅-0.5~+1.5% + 振幅<3% + 距MA20≤8% |
| 第二层 | **RSI拐头** | RSI(6)今>昨 + 昨<35 + 距MA20≤8% + DIF收敛 + 未金叉 |
| 第三层 | **MACD金叉** | 涨幅3-9.5% + MA20上 + MACD金叉≤5天 + 放量>1.5x |

## 📊 核心策略

### 五指标共振模型

| 指标 | 看什么 | 信号顺序 | 动作 |
|------|--------|----------|------|
| MA(20) | 方向 | 前提 | 站上=多头，跌破=空头 |
| RSI(6) | 衰竭 | ① 最早 | 拐头+<35 → 加入自选 |
| KDJ(9,3,3) | 时机 | ② 次早 | 低位金叉 → **轻仓试（第一个买点）** |
| MACD(12,26,9) | 确认 | ③ 最晚 | 金叉 → 加仓确认 |
| 成交量 | 真伪 | 贯穿 | 放量>1.5x → 真金白银 |

**核心原则**：KDJ 金叉是动手信号，MACD 金叉是加仓信号。不等 MACD 才入场。

### 决策树（9 条全过才做）

```
1. 大盘前一日跌幅 > 1.5%           → 不做
2. 大盘已连涨 ≥ 2天，今天是第3天     → 不做
3. 当天金叉+零轴上 标的 < 2只       → 不做
4. 最佳标的 PE > 80                 → 不做
5. 最佳标的 量比 > 3x               → 不做
6. 最佳标的 KDJ高位钝化(≥4天K>80)    → 不做
7. 最佳标的 KDJ未金叉               → 不做
8. 持仓有需紧急处理的               → 优先处理持仓
9. 持仓已满或仓位 > 50%             → 不做新开仓
```

**全部通过 → 做。任意一条不通过 → 不做。**

### 四类结论

每只票必须归入以下四类之一：

1. **可直接试仓买入** — 逻辑清晰 + 环境支持 + 买点合适
2. **条件满足才可买入** — 有逻辑但买点不成熟，需等特定条件
3. **只可观察，不可买** — 有逻辑但风险收益比差
4. **明确不买** — 不符合标准

### 风控铁律

| 规则 | 条件 | 操作 |
|------|------|------|
| 单票硬止损 | 浮亏 ≥ 8% | 无条件清仓 |
| 组合日预警 | 单日浮亏 ≥ 5% | 不再开新仓 |
| 组合周强制 | 周浮亏 ≥ 10% | 清仓冷却 3 天 |
| 单票上限 | — | 20% |
| 单行业上限 | — | 40% |

## ⚙️ 数据源

所有数据来自 [腾讯财经](https://finance.qq.com/) HTTP 接口：

- **K 线数据**：`web.ifzq.gtimg.cn` — 日线 OHLCV
- **实时行情**：`qt.gtimg.cn` — 价格、涨跌幅、市值等
- **股票列表**：`smartbox.gtimg.cn` — 全市场 A 股代码

无需申请 API Key，无频率限制（建议适当间隔避免被封）。
