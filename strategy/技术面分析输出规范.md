# 技术面分析输出规范 V1.0

> 所有标的推荐、个股分析、老板点名询问，统一使用此格式输出。

---

## 一、输出格式模板

```
📈 石昊 | {股票名称}({代码}) 技术面分析

{股票名称}({代码}) | 现价 {current_price}

1️⃣ 当前股价位置
{均线位置判断，每条一行}
{趋势判断}
{技术指标信号}

2️⃣ 关键位置
压力位: {price}（{来源说明}）
支撑位: {price}（{来源说明}）
MA20: {price}（趋势线）

3️⃣ 是否适合现在介入？
{✅/❌} {结论}
原因：
- {理由1}
- {理由2}
- {理由3}

4️⃣ 如果要买，价格方案
| 项目 | 价格 | 说明 |
|------|------|------|
| 触发价 | xxx | 什么条件触发买入 |
| 建议买入价 | xxx | 建议入场价格 |
| 止盈目标价 | xxx | 第一止盈位 |
| 止损价 | xxx | 硬止损或技术止损 |
| 盈亏比 | x.x:1 | ≥2:1才推荐 |

📊 结论：{一句话总结}
```

---

## 二、各部分填写规范

### 1️⃣ 当前股价位置

**必须包含：**
- MA5/MA10/MA20 站上/跌破状态（用✅/❌标记）
- 均线排列状态（多头/空头/纠缠）
- KDJ 状态（金叉/死叉 + K/D值 + 是否超买超卖）
- MACD 状态（金叉/死叉 + 零轴位置）

**格式示例：**
```
✅ 站上MA5/MA10/MA20，均线多头排列
✅ KDJ金叉持续(K=68 D=51)
❌ MACD死叉持续（DIF<DEA）
```

### 2️⃣ 关键位置

**必须包含：**
- 压力位：20日高点 / 前高 / 布林上轨（取最近最相关的）
- 支撑位：20日低点 / MA20 / 整数关口
- 每个位置标注来源

**格式示例：**
```
压力位: 159.05（20日高点）
支撑位: 120.00（20日低点）
MA20: 135.02（趋势线）
```

### 3️⃣ 是否适合现在介入

**必须给出明确结论（✅可以 / ❌不可以 / ⚠️等条件）**

**理由必须包含：**
- 技术面信号是否共振
- 盈亏比是否达标（≥2:1）
- 是否有明显风险信号

**格式示例：**
```
❌ 不建议现在介入
原因：
- MACD死叉，与技术面矛盾
- J值超买(104.65)，短期可能回调
- 盈亏比1.63:1 < 2:1（不达标）
```

### 4️⃣ 价格方案

**所有"可以买"或"条件满足才买"的标的必须填写价格方案。**

| 字段 | 说明 | 计算方式 |
|------|------|----------|
| 触发价 | 什么价格/条件触发买入 | 突破压力位 / 回踩支撑位 |
| 建议买入价 | 建议入场价格 | 触发价确认后介入 |
| 止盈目标价 | 第一止盈位 | 20日高点 / 前高 / 压力位 |
| 止损价 | 亏损离场价 | min(成本-8%, 技术止损位) |
| 盈亏比 | 盈利空间/亏损空间 | (止盈-买入)/(买入-止损) ≥ 2:1 |

**盈亏比计算示例：**
```
买入价: 7.75
止盈: 8.50 → 盈利空间 = (8.50-7.75)/7.75 = 9.7%
止损: 7.06 → 亏损空间 = (7.75-7.06)/7.75 = 8.9%
盈亏比 = 9.7/8.9 = 1.09:1 ❌ 不达标

买入价: 7.75
止盈: 10.00 → 盈利空间 = (10.00-7.75)/7.75 = 29.0%
止损: 7.06 → 亏损空间 = 8.9%
盈亏比 = 29.0/8.9 = 3.26:1 ✅ 达标
```

---

## 三、结论规范

最后一句话必须是以下之一：

| 结论 | 含义 | 格式 |
|------|------|------|
| ✅ 可以介入 | 满足全部条件，盈亏比达标 | `结论：可以介入，盈亏比x.x:1` |
| ⚠️ 等条件 | 有信号但未完全确认 | `结论：等{条件}再考虑` |
| ❌ 不建议 | 技术面不支持或盈亏比不够 | `结论：不建议介入，原因：{一句话}` |

---

## 四、禁止事项

- ❌ 不带价格方案说"可以买"
- ❌ 不说"可以关注"这种模糊表达
- ❌ 不列一堆理由不给结论
- ❌ 不算盈亏比就推荐
- ❌ 盈亏比<2:1还说"可以介入"

---

## 五、适用场景

| 场景 | 是否用此格式 |
|------|-------------|
| 老板点名问某只票 | ✅ 必须 |
| 盘后扫描推荐标的 | ✅ 必须 |
| 博弈群个股分析 | ✅ 必须 |
| 持仓止盈止损跟踪 | ✅ 必须（带当前盈亏比） |
| 观察池标的异动提醒 | ✅ 必须 |
| 全市场扫描汇总 | 可用简化版（表格+关键信号） |

## 六、简化版（全市场扫描汇总时）

当一次性输出多只标的时，可用表格简化：

```
🎯 全市场扫描结果

| 标的 | 现价 | 涨幅 | 得分 | 盈亏比 | 信号 | 结论 |
|------|------|------|------|--------|------|------|
| XX(000001) | 10.00 | +5% | 4/5 | 2.3:1 ✅ | 多头+MACD金叉 | ✅可介入 |
| YY(000002) | 20.00 | +3% | 3/5 | 1.5:1 ❌ | KDJ金叉 | ⚠️等确认 |
```

简化版只对得分最高的1-2只展开详细分析（用完整格式）。
