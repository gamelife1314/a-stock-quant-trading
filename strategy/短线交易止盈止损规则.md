## 一、入场条件（引用现有决策树）

### 1.1 入口决策树（9条，全部通过才做）

```
1. 大盘前一日跌幅 > 1.5%           → 不做
2. 大盘已连涨 ≥ 2天，今天是第3天     → 不做
3. 当天金叉+零轴上 标的 < 2只       → 不做
4. 最佳标的 PE > 80                 → 不做
5. 最佳标的 量比 > 3x               → 不做
6. 最佳标的 KDJ高位钝化(K>80连续≥4天) → 不做
7. 最佳标的 KDJ未金叉(K仍在D下方)    → 不做
8. 持有标的中有需要紧急处理的        → 优先处理持仓，不做新开仓
9. 持仓已满或仓位 > 50%             → 不做新开仓
```

**全部通过 → 做。任意一条不通过 → 不做。**

### 1.2 入场信号优先级

| 信号 | 触发动作 | 仓位 |
|------|----------|------|
| RSI(6)拐头↑ + 昨日<35 | 加入自选，不买 | 0% |
| KDJ低位金叉 + 站上MA20 + 放量>1.3x | **轻仓试**（第一个买点） | 5-10% |
| MACD金叉确认（KDJ已金叉） | **加仓** | 加至15-20% |

### 1.3 标的硬性排除

- ST / *ST / 退市股 → 直接排除
- PE亏损 或 PE>200 → 排除
- 市值<30亿 或 >5000亿 → 排除
- 涨停板（≥9.5%）→ 不推荐
- 次新股（上市<60天）→ 不纳入

## 二、止损规则（三层止损体系）

### 2.1 🔴 硬止损（无条件执行）

| 规则 | 触发条件 | 操作 | 说明 |
|------|----------|------|------|
| **单票硬止损** | 单票浮亏 ≥ **8%** | 无条件全部卖出 | 王林风控铁律第一条 |
| **单日硬止损** | 单票日内跌幅 ≥ **6%** | 无条件全部卖出 | 防止日内崩盘 |
| **跳空硬止损** | 次日开盘跳空低开 ≥ **5%** | 集合竞价挂跌停出 | 不抱幻想 |

> ⚠️ **硬止损是铁律，不允许任何理由违反。** 到了就砍，不讨论、不等、不看。

### 2.2 🟡 技术止损（指标触发）

| 级别 | 触发条件 | 操作 | 仓位变化 |
|------|----------|------|----------|
| **一级预警** | 跌破 MA5 + 缩量 | 减仓 1/3 | 留 2/3 |
| **二级预警** | 跌破 MA10 + KDJ死叉 | 减仓 1/2 | 留 1/2 |
| **三级止损** | 跌破 MA20 + MACD死叉 | **全部卖出** | 清仓 |
| **RSI止损** | RSI(6) > 85 + 冲高回落 > 3% | 减仓 1/2 | 留 1/2 |
| **量价止损** | 放量 > 5x + 滞涨（涨<2%） | **全部卖出** | 清仓 |

### 2.3 🟠 时间止损（避免资金闲置）

| 规则 | 触发条件 | 操作 |
|------|----------|------|
| **T+3 止损** | 买入后第3个交易日仍未盈利 | 减仓 1/2 |
| **T+5 止损** | 买入后第5个交易日浮亏 > 3% | **全部卖出** |
| **横盘止损** | 连续5个交易日涨跌幅在 ±2% 以内 | 减仓 1/2，腾资金 |
| **金叉失效止损** | MACD金叉后5个交易日内未出现 > 3% 的阳线 | **全部卖出** |

### 2.4 止损执行原则

1. **止损不隔夜**：盘中触发技术止损 → 当日执行，不等明天
2. **止损不补仓**：触发止损意味着判断错误，补仓是错上加错
3. **止损后冷却**：同一标的止损后 3 个交易日内不重新买入
4. **止损记录**：每次止损写入 `memory/` 日志，标注原因

## 三、止盈规则（分批止盈 + 移动止盈）

### 3.1 分批止盈比例

| 达到目标 | 操作 | 剩余仓位 | 逻辑 |
|----------|------|----------|------|
| 浮盈 **+5%** | 卖出 **1/3** | 留 2/3 | 锁定部分利润，降低持仓成本 |
| 浮盈 **+10%** | 卖出 **1/3**（累计卖出 2/3） | 留 1/3 | 本金基本收回 |
| 浮盈 **+15%** 以上 | 卖出最后 1/3 | 清仓 | 短线不贪，15%足够 |

### 3.2 移动止盈（跟踪止盈）

| 止损线上移规则 | 触发条件 | 新止损线 |
|---------------|----------|----------|
| **初始止损** | 开仓时 | MA20 或 成本价-8%（取低者） |
| **第一上移** | 浮盈 ≥ 5% | 移动至 **成本价**（保本） |
| **第二上移** | 浮盈 ≥ 8% | 移动至 **MA5**（跌破即出） |
| **第三上移** | 浮盈 ≥ 12% | 移动至 **前日收盘价 - 3%** |

> 核心：**永远不要让盈利的持仓变成亏损。** 浮盈超过 5%，止损线上移到成本价。

### 3.3 加速止盈（卖方力量信号）

| 信号 | 操作 |
|------|------|
| RSI(6) > 85 | 减仓 1/2（超买见顶） |
| KDJ J值 > 100 然后回落跌破 100 | 减仓 1/2 |
| 高位放量滞涨（量>3x，涨<2%） | 减仓 2/3 |
| 高位长上影线（上影 > 实体 2x） | 减仓 1/2 |
| 高位连续 3 根阳线后首根阴线 + 放量 | **全部卖出** |
| MACD 红柱连续 3 天缩短 | 减仓 1/2 |
| MACD 高位死叉（DIF 下穿 DEA + DIF>0） | **全部卖出** |

### 3.4 止盈卖出优先级

```
1. 先卖浮盈最大的 → 锁定最丰厚利润
2. 再卖信号最弱的 → 剔除风险最大的
3. 最后卖最确定的 → 保留最好的
```

## 四、仓位管理

### 4.1 总仓位限制

| 市场环境 | 总仓位上限 | 说明 |
|----------|-----------|------|
| S级（可积极出手） | **50%** | 单行业≤20%，单票≤15% |
| A级（参与但筛选） | **40%** | 单行业≤15%，单票≤10% |
| B级（轻仓试错） | **20%** | 单行业≤10%，单票≤8% |
| C级（观察为主） | **10%** | 单票≤5%，不超2只 |
| D级（不主动出手） | **0%** | 不持有新仓，处理旧仓 |

### 4.2 单票仓位阶梯

| 阶段 | 仓位 | 条件 |
|------|------|------|
| 试仓（首仓） | **5-8%** | KDJ低位金叉+站上MA20+放量 |
| 加仓一 | **+5%**（累计10-13%） | MACD金叉确认 |
| 加仓二 | **+5%**（累计15-18%） | 浮盈>5%+MACD红柱放大 |
| 最大持仓 | **20%** | 单票上限，任何情况不超过 |

### 4.3 加仓规则

| 规则 | 说明 |
|------|------|
| **不在亏损时加仓** | 浮亏状态下禁止加仓，永远不摊低成本 |
| **加仓间距** | 两次加仓之间至少间隔 1 个交易日 |
| **加仓确认** | 加仓当日必须放量 > 1.3x |
| **加仓上限** | 单票累计不超过 20% |

### 4.4 减仓/清仓队列

触发技术止损/止盈时，按以下顺序执行：

```
1. 先减信号最弱/浮亏最大的
2. 再减浮盈已达目标的
3. 最后处理核心持仓
```

## 五、持仓周期

### 5.1 周期定义

| 类型 | 持有时间 | 目标收益 | 适用场景 |
|------|----------|----------|----------|
| **超短线** | 日内（T+0） | 1-3% / 笔 | 高确定性、强放量突破 |
| **短线** | 1-3 个交易日 | 3-8% / 笔 | KDJ+ MACD 金叉共振 |
| **短持** | 3-10 个交易日 | 8-15% / 笔 | 主线趋势明确 |

### 5.2 持仓时间纪律

| 规则 | 说明 |
|------|------|
| **默认短线** | 开仓时默认 1-3 个交易日 |
| **T+3 评估** | 第 3 个交易日强制评估：留 or 走 |
| **T+5 强制** | 第 5 个交易日仍未达目标且无新信号 → 清仓 |
| **周末空仓原则** | 周五收盘前未浮盈 > 3% 的仓位，能清则清 |

### 5.3 超短线特别规则（日内）

| 规则 | 说明 |
|------|------|
| 开盘 30 分钟内不追高 | 等开盘情绪释放 |
| 尾盘 30 分钟（14:30 后）可开仓 | 博弈次日溢价 |
| 日内涨幅 > 5% 后不加仓 | 追高风险可控 |
| 日内止损 | 开仓价 - 3% 硬止损 |

## 六、最大回撤控制

### 6.1 组合级别

| 级别 | 触发条件 | 操作 |
|------|----------|------|
| 🟡 **日预警** | 组合单日浮亏 ≥ **5%** | 当日不再开新仓，次日减仓至半仓以下 |
| 🟠 **周预警** | 组合周浮亏 ≥ **7%** | 减仓至 20% 以下 |
| 🔴 **周强制** | 组合周浮亏 ≥ **10%** | **清仓所有短线仓位**，冷却 3 个交易日 |
| 🔴 **月强制** | 组合月浮亏 ≥ **15%** | **停止交易**，复盘全部操作后再决定 |

### 6.2 单票级别

| 级别 | 触发条件 | 操作 |
|------|----------|------|
| 🟡 预警 | 单票浮亏 ≥ 5% | 禁止加仓，准备减仓 |
| 🟠 减仓 | 单票浮亏 ≥ 6% | 减仓 1/2 |
| 🔴 硬止损 | 单票浮亏 ≥ **8%** | **无条件清仓** |

### 6.3 连续亏损熔断

| 连续亏损笔数 | 操作 |
|-------------|------|
| 连续 3 笔亏损 | 暂停开新仓 1 个交易日，复盘原因 |
| 连续 5 笔亏损 | **停止交易 3 个交易日**，全面复盘 |
| 单周亏损 ≥ 总资金 10% | **停止交易 1 周** |

### 6.4 单行业集中度控制

| 限制 | 上限 |
|------|------|
| 单行业总仓位 | ≤ **40%**（王林风控铁律） |
| 单行业单票数量 | ≤ **3 只** |
| 同概念方向仓位 | ≤ **30%**（避免共振下跌） |

## 七、执行纪律

### 7.1 铁律（不可违反）

```
1. 硬止损 8% 不可违反          — 触之即砍
2. 组合周回撤 10% 不可违反       — 触之即清
3. 单票仓位 20% 不可违反        — 触之即减
4. 不在亏损时补仓               — 永远不摊低成本
5. 止损后 3 天不重新买入同一标的   — 强制冷静期
```

### 7.2 每日必做

- [ ] 盘前：检查持仓止损/止盈线是否上移
- [ ] 盘中：触发止损立即执行，不等收盘
- [ ] 收盘：所有持仓评估，更新止损线
- [ ] 记录：止损/止盈操作写入 `memory/YYYY-MM-DD.md`

### 7.3 盘后复盘模板

```
【今日操作】
- 开仓：X只，仓位X%
- 平仓：X只，盈亏+X%
- 止损触发：X次

【持仓状态】
| 标的 | 仓位 | 成本 | 现价 | 浮盈 | 止损线 | 持有天数 |
|------|------|------|------|------|--------|----------|

【回撤监控】
- 今日组合浮亏：X%
- 本周累计浮亏：X%
- 连续亏损笔数：X笔
- 当前总仓位：X%
```

## 八、规则优先级

当多个规则同时触发时，按以下优先级执行：

```
1. 硬止损（单票-8%、日-6%、跳空-5%）        ← 最高
2. 组合回撤控制（日-5%、周-10%）            ← 次高
3. 技术止损（MACD死叉、跌破MA20）           ← 第三
4. 时间止损（T+5、横盘5天）                 ← 第四
5. 止盈规则（分批止盈、移动止盈）            ← 第五
```

> 止损永远优先于止盈。**保住本金是第一要务。**
