# A股短线量化交易

> **版本**: V1.0 | **日期**: 2026-06-19

## 描述

A 股短线量化交易完整策略包。包含五指标共振模型、双轨扫描策略、决策树系统、止盈止损规则，以及配套的 Python 全市场扫描脚本。

**零外部依赖**：所有数据接口使用腾讯财经 HTTP，技术指标纯 Python 计算，`python3 scripts/scanner.py` 即可运行。

## 适用场景

- 盘后全市场扫描（三层信号链路：地量止跌 → RSI拐头 → MACD金叉）
- 盘前决策判断（环境评级 + 决策树 9 条 → 做 / 不做）
- 持仓标的跟踪（技术面体检 + 止盈止损触发检测）
- 个股深度分析（五指标矩阵 + 四类结论输出）

## 使用方式

```bash
# 运行全市场扫描（零依赖）
python3 scripts/scanner.py

# 输出：控制台打印三层扫描结果 + 保存 JSON 到 data/scan_results.json
```

## 核心策略摘要

### 五指标共振模型

| 指标 | 看什么 | 信号顺序 | 动作 |
|------|--------|----------|------|
| MA(20) | 方向 | 前提 | 站上=多头，跌破=空头 |
| RSI(6) | 衰竭 | ① 最早 | 拐头+<35=加入自选 |
| KDJ(9,3,3) | 时机 | ② 次早 | 低位金叉=**轻仓试（第一个买点）** |
| MACD(12,26,9) | 确认 | ③ 最晚 | 金叉=加仓确认 |
| 成交量 | 真伪 | 贯穿 | 放量>1.5x=真金白银 |

**入场策略**：KDJ 金叉是动手信号，MACD 金叉是加仓信号。不等 MACD 才入场。

### 决策树（10 条全过才做）

```
1. 大盘前一日跌幅 > 1.5%           → 不做
2. 大盘已连涨 ≥ 2天，今天是第3天     → 不做
3. 当天金叉+零轴上 标的 < 2只       → 不做
4. 最佳标的 PE > 80                 → 不做
5. 最佳标的 量比 > 3x               → 不做
6. 最佳标的 KDJ高位钝化(≥4天K>80)    → 不做
7. 最佳标的 KDJ未金叉               → 不做
8. 持仓有需紧急处理的               → 优先处理持仓
9. 持仓已满或仓位 > 50%             → 不做新开仓
10. 盈亏比 < 2:1                    → 不做
```

**盈亏比 = 预期盈利空间 / 最大可接受亏损 ≥ 2:1**

```
盈亏比 = (技术止盈目标位 - 买入价) / (买入价 - 止损位)
```

示例：买入 30 元，止损 27.6 元(-8%)，止盈目标 35 元 → 盈亏比 = 5/2.4 = 2.08:1 ✅

盈亏比不足 2:1 = 赔率不够 = 不值得冒险。

全部通过 → 做。任意一条不通过 → 不做。

### 双轨扫描策略

- **轨一：主线方向精选** — 方向热度 → 方向内找涨幅 3-7% + 指标共振
- **轨二：独立突破扫描** — 全市场扫 MA20 突破 + MACD 翻多 + 放量 > 1.5x（含周期反转股）
- **优先级**：主线方向 > 独立突破

### 个股输出结构（含价格方案）

每只票输出必须包含：
- 方向归属、逻辑强度、个股地位、当前位置判断、买点判断、风险点
- **操作结论**（四类之一）
- 结论理由、条件触发
- **价格方案**：触发价 / 建议买入价 / 止盈目标价 / 止损价 / 盈亏比（≥2:1）

1. **可直接试仓买入** — 逻辑清晰 + 环境支持 + 买点合适
2. **条件满足才可买入** — 有逻辑但买点不成熟，需等特定条件
3. **只可观察，不可买** — 有逻辑但风险收益比差
4. **明确不买** — 不符合标准

### 排除标准

| 类型 | 规则 |
|------|------|
| 硬性排除 | ST/\*ST、涨停(≥9.5%)、PE 亏损或 >200、市值 <30 亿或 >5000 亿 |
| 软性降级 | 市值 30-50 亿、PE >100、次新股(<60 天) |
| 涨幅 | 最佳 3-7%，偏高 7-9.5%，禁止 ≥9.5% |

### 风控铁律

- **单票硬止损**: 浮亏 ≥8% 无条件清仓
- **组合日预警**: 单日浮亏 ≥5% 不再开新仓
- **组合周强制**: 周浮亏 ≥10% 清仓冷却 3 天
- **单票上限**: 20%
- **单行业上限**: 40%

## 数据脚本

`scripts/scanner.py` — 三层信号链路全市场扫描，零外部依赖。

**数据源**：腾讯财经 HTTP 接口（K 线 + 实时行情）

**三层扫描**：

1. **地量止跌** — 量 <20 日均量 60% + 涨跌幅 -0.5~+1.5% + 振幅 <3% + 距 MA20≤8%
2. **RSI 拐头** — RSI(6) 今>昨 + 昨<35 + 距 MA20≤8% + DIF 收敛 + 未金叉
3. **MACD 金叉确认** — 涨幅 3-9.5% + MA20 上 + MACD 金叉 ≤5 天 + 放量 >1.5x

**内置技术指标**：MA, RSI, KDJ, MACD, 布林线（纯 Python 实现）

**依赖**：零。仅使用 Python 标准库（json, math, time, os, urllib, ssl, csv, io, datetime）。

## 环境评级体系

| 级别 | 含义 | 仓位上限 |
|------|------|----------|
| S 级 | 可积极出手 | 50% |
| A 级 | 可以参与，需筛选 | 40% |
| B 级 | 轻仓试错 | 20% |
| C 级 | 观察为主 | 10% |
| D 级 | 原则上不主动出手 | 0% |

## 信号时效性规则

| 金叉时间 | 信号强度 | 处理 |
|----------|----------|------|
| 当天(Day 0) | 最强 | 优先推荐 |
| 1-2 天 | 有效窗口 | 标注"窗口有限" |
| 3-5 天 | 衰减 | 降级处理 |
| >5 天 | 失效 | 不推荐 |
